## Fase atual

**Curto prazo — COMPLETO** (v0.0.1 → v0.6.0): widget tree, layout, display-list, focus, text editing, animation, theme tokens.

**Migração de ownership do desktop — COMPLETO** (v0.7.0 → v0.7.3): a sessão de desktop, window manager e built-in apps que viviam no fallback in-tree do CapyOS migraram para `src/desktop/`, `src/window/`, `src/apps/` neste repo, agora publicados como o módulo `org.capyos.ui.desktop-session` (ABI `capy-ui-desktop-session` v1) ao lado do `org.capyos.ui.widget-core`.

**Médio prazo — COMPLETO** (v0.8 → v1.0). v0.8 (compositor diff), v0.9 (texto/fontes host), v0.10 (input plumbing), v0.11 (a11y tree), v0.13 (i18n + RTL), v0.14 (theming UX), v0.15 (performance tier 1) e **v1.0 (congelamento)** entregues. As únicas pendências são `v0.7` (adapter) e `v0.12` (shell), ambas gated por etapas externas do CapyOS e ambas metadata-only sobre o ABI 1.x já congelado.

**Longo prazo — fase 1.x COMPLETA + 2.0–2.3 entregues**. v1.1–1.10 + v2.0–2.3 entregues (`damage tracking, GPU translator, rich animation, gesture engine, multi-display + DPI, drag and drop, slab allocator, rich IME, transforms 2D, state serialization, plugin system + SDK público, advanced widgets enum-only, virtualization plumbing, undo/redo per-context`). 1.x em LTS ≥12m. Próximo marco: v2.4 theme packs.
