## Objetivo

Integração com taskbar, dock, notificações e foco entre janelas. **Só prossegue se Etapa 6 autorizar.**

## Escopo CapyUI

- Tipos aditivos no enum `capy_widget_type`:
  - `CAPY_WIDGET_NOTIFICATION`
  - `CAPY_WIDGET_TRAY_ICON`
- Estes tipos só fazem sentido com shell real; em testes host puros são modelo retido apenas (sem renderização específica).
- Default style para cada tipo (configurável via theme tokens).

## Escopo CapyOS

- Implementação concreta de shell:
  - Taskbar mostra aplicações abertas com label e ícone.
  - Dock recebe drop (preparação para v1.6 drag&drop completo).
  - Notificações empilham com timeout configurável.
  - Política de foco entre janelas (Z-order, alt-tab).
- App launcher integrado.
- Configuração de áreas de tela (taskbar top/bottom/left/right).

## Testes obrigatórios (CapyOS)

1. **Aplicação CapyUI aparece na taskbar** com label correto.
2. **Notificação emitida via API** aparece sem bloqueio do thread de UI.
3. **Tray icon clicável** dispara callback do app.
4. **Alt-Tab cicla janelas** sem perda de foco interno (CapyUI mantém widget focado).
5. **Drop em dock** dispara evento abstrato no app destino.
