## Objetivo

Reconhecedor de gestos determinístico: tap, double-tap, long-press, swipe (4 direções), pinch, rotate, drag. Mesma entrada sintética → mesma saída.

## Escopo CapyUI

- Enum:
  ```c
  enum capy_gesture_kind {
    CAPY_GESTURE_NONE = 0,
    CAPY_GESTURE_TAP,
    CAPY_GESTURE_DOUBLE_TAP,
    CAPY_GESTURE_LONG_PRESS,
    CAPY_GESTURE_SWIPE_LEFT,
    CAPY_GESTURE_SWIPE_RIGHT,
    CAPY_GESTURE_SWIPE_UP,
    CAPY_GESTURE_SWIPE_DOWN,
    CAPY_GESTURE_PINCH_IN,
    CAPY_GESTURE_PINCH_OUT,
    CAPY_GESTURE_ROTATE_CW,
    CAPY_GESTURE_ROTATE_CCW,
    CAPY_GESTURE_DRAG
  };
  ```
- Struct:
  ```c
  struct capy_gesture {
    enum capy_gesture_kind kind;
    struct capy_ui_point start;
    struct capy_ui_point end;
    int32_t magnitude;        /* pixels para swipe, % para pinch, graus*256 para rotate */
    uint32_t duration_ms;
  };
  ```
- Recognizer:
  ```c
  struct capy_gesture_recognizer;  /* opaco */

  void capy_gesture_recognizer_init(struct capy_gesture_recognizer *r,
                                    void *buf, uint32_t buf_size);
  int capy_gesture_feed(struct capy_gesture_recognizer *r,
                        const struct capy_widget_event *ev);
  int capy_gesture_poll(struct capy_gesture_recognizer *r,
                        struct capy_gesture *out);
  ```
- Constantes documentadas: thresholds de distância, tempo, ângulo.

## Testes obrigatórios

1. **Tap único** com `dist < threshold` e `t < 200ms`.
2. **Double-tap** dois taps com gap < 300ms.
3. **Long-press** sem movimento por > 500ms.
4. **Swipe** velocidade mínima + direção dominante.
5. **Pinch** dois dedos aproximando/afastando.
6. **Rotate** dois dedos rotacionando ao redor de centro.
7. **Determinismo:** mesma sequência de events → mesmo gesto.
8. **Reentrant feed** não corrompe estado.

## Notas

- Determinístico: thresholds documentados, comportamento previsível.
- Memory: recognizer usa buffer fornecido pelo caller (sem malloc).
- Host fornece touch events via v0.10 plumbing.
